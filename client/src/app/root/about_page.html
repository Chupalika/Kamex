About
The main purpose of this website is to provide an accessible platform for osu! players to host tournaments. It is very new and still very much in development, so please excuse its barebones-ness :^)
Features
Here are some of the cool features currently on Kamex! * Players can register for your tournament by authenticating through osu! and clicking the register button. No need to worry about Google forms! * Players can also link their Discord account so that it shows up on their card for reference. This can also be used by the bot for ping reminders and such! * Pages to view details about participants, mappools, and matches * Various tools to manage tournament logistics: * Edit settings like registration rules, description, links * Add/edit/remove things like players, teams, staff members, rounds, etc. * Edit staff roles with different permissions, such as editing mappools and matches * Statistics page displays score data for each tourney round in table form with various options * Discord bot which can log some staff activity and announce reminders for upcoming matches * Customizable primary/accent colors and (mappool slot) category colors
Upcoming Features
Here are some of upcoming features that I am planning to add eventually (subject to change)! * Team creation: A feature where tourney registrants can create teams and invite other registrants and edit its name and image. * Scoresheet feature: Create scoresheets for your teams to track scores. There will be settings to customize things like layout, permissions, and highest/avg scores. This is actually already partially implemented in the form of tournament stats. * Referee tool: A feature that will simplify match refereeing, with buttons to invite players/teams, change maps, start/abort, etc. and keep track of picks/bans and score. Will also auto submit match results and progression to Kamex! * Match progression: Records things like warmups, bans, rolls, picks, and possibly tourney-specific custom things like, say, trap cards. * Reschedules: Request reschedules for your match. If opponent(s) accept, it will be updated, and is also useful as confirmation rather than rely on screenshots of DMs. * Discord bot: Already partially implemented, but in the future will also announce reschedules and matches in progress. I may also have it be possible to ref using the bot. * Various small improvements/additions, like table view in mappool and participant pages, auto suggest match schedules, bracket.json generation, custom fonts, custom category icons. * Design improvements (as you could probably tell I am not a designer LOL) When will you implement [feature]? * Unfortunately, I have started a full time job since starting this project, so it's very difficult to find time and motivation to work on it 🥀 so I apologize for the slow updates.
